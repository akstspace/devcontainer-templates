## Python Development Container

This template provides a complete Python development environment with:

- **Python** - Multiple version options (3.12, 3.11, 3.10)
- **pip** - Python package installer
- **Poetry** - Optional dependency management tool
- **Git** - Version control
- **Zsh & Oh My Zsh** - Enhanced shell experience

### Features

- Pre-configured Python extension for VS Code
- Pylance for enhanced IntelliSense
- Ruff for fast linting and formatting
- Automatic formatting on save

### Getting Started

After creating your dev container:

1. Create a virtual environment: `python -m venv .venv`
2. Activate the environment: `source .venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Start coding!

### Customization

You can customize the template by modifying the `devcontainer.json` file in your `.devcontainer` folder.
