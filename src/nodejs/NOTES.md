## Node.js Development Container

This template provides a complete Node.js development environment with:

- **Node.js** - Multiple version options (20, 18, 16)
- **npm** - Included with Node.js
- **pnpm** - Optional package manager
- **Git** - Version control
- **Zsh & Oh My Zsh** - Enhanced shell experience

### Features

- Pre-configured ESLint and Prettier extensions for VS Code
- Automatic formatting on save
- Common development utilities

### Getting Started

After creating your dev container:

1. Initialize your project: `npm init -y`
2. Install dependencies: `npm install`
3. Start coding!

### Customization

You can customize the template by modifying the `devcontainer.json` file in your `.devcontainer` folder.
